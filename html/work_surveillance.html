Web Surveillance

Interactive Website

November - December 2016

This project was created as a way to make the user realize how much information they are willingly providing when using the internet. The web is free for us to use, however the currency is personal information. We give pieces of ourselves in order for a world wide access, such as signing up for social media accounts, searching for products online, and more. Also, we tend to not even care about the type of information we are putting out into this digital space or realize what kind of impact this could have on us. This project explores the lack of privacy in an online environment.

Technical - HTML5, CSS3, jQuery
